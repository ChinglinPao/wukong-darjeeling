To enable the protobuf and python rpc, you need to download and install:

1) Protobuf library
https://github.com/google/protobuf

2) Proto RPC channel implementation
https://code.google.com/p/protobuf-socket-rpc/downloads/detail?name=protobuf.socketrpc-1.3.2-py2.6.egg&can=2&q=

To use python mock test, please install pymock
3) Pymock
http://theblobshop.com/pymock/


4) Realize binary program
Install pythonhosted.org/PyInstaller/
